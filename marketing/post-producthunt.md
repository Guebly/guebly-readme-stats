# Product Hunt Launch — Guebly ReadMe Stats

---

## Tagline
Dynamic GitHub stats cards for your README — 14 types, 70+ themes, open source

## Description

**Guebly ReadMe Stats** generates beautiful, dynamic SVG cards for your GitHub profile README — updated automatically on every request.

### What you can generate:
- **Stats Card** — commits, PRs, issues, stars, rank
- **Activity Graph** — sparkline of monthly contributions
- **Contributions Heatmap** — GitHub-style calendar
- **Tech Stack** — languages grouped by category
- **Compare Users** — side-by-side dev comparison
- **Top Languages, Streak, Social, Trophy** — and 6 more

### Why it's different:
- **70+ themes** including 10 exclusive designs
- **Custom colors** via URL parameters
- **Visual generator** with live preview
- **Download PNG** and Instagram Story (9:16)
- **Embed mode** for Notion, portfolios, blogs
- **Self-hostable** on Vercel free tier
- **Open source** (MIT license)

### How it works:
1. Go to readme.stats.guebly.com.br
2. Pick a card type and theme
3. Enter your GitHub username
4. Copy the Markdown/HTML
5. Paste in your README — done!
